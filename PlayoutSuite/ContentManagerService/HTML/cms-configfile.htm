Content Manager Service (CMS)

Config File

Application

<add key="TmpFolder" value="D:\HYBRIDDEMO\DATA\tmp\"/>

TmpFolder is the folder used to generate the files.


<add key="DataFolder" value="D:\HYBRIDDEMO\DATA\"/>

DataFolder specifies the folder which contains the data folders required by CMS.
See Folder Structures

Slides

<add key="broadcastdelay" value="5"/>

broadcastdelay key specifies in seconds the default duration of a slide.

 

<add key="radiovislink" value="http://tech.ebu.ch"/> 
<add key="radiovistxt" value="Stay up to date: http://tech.ebu.ch/">

radiovislink and radiovistxt keys are the default values of slides, if not specified in slides.

 

<add key="slidecartprefix" value="%prefix%"/>

The slide cart will loaded with every slide beginning by %prefix%

 

Input

<add key="InputMethod" value="#InputMethod"/>

#InputMethod is either WCFstrong>WCF HTTP

WCF is used when you want to synchronize the CMS with the EBU Radio Production Platform.

HTTP is used when you want to send input values to update the variables of the CMS. CMS.
See HTTP updates parameters.

 

Output

Stomp

<add key="stompServer" value="#stompServer"/> 
<add key="stompPort" value="#stompPort"/> 
<add key="stompLogin" value="#stompLogin"/> 
<add key="stompPasscode" value="#stompPasscode"/> 
<add key="stompHttpUrl" value="#stompHttpUrl"/>"/>

#stompServer: The server address
#stompPort: The server port
#stompLogin: The server login
#stompPasscode: The server password
#stompHttpUrl: The url to the folder where the generated image is available from receivers.


<add key="stompTopicCount" value="#numberOfTopics"/>
<add key="stompTopic#N" value="#topic"/>

#topic: example /topic/dab/4e1/4fff/4000/0/

FTP

<add key="ftpCount" value="g>#ftpCount"/>

#ftpCount: It's: It's the number of ftp parameter's set.


<add key="ftpServerg>#N" value="ftp://yourserver/folder/"/> 
<add key="ftpUser#N" value="ftpuser"/> 
<add key="ftpPwd#N" value="ftppassword"/> 
<add key="ftpAlone#N" value="#ftpAlone"/> 
<add key="ftpMinPeriod#N" value="#ftpMinPeriod"/>"/>

#N represents an integer from 1 to N for each parameter's set

#ftpAlone: Used for DAB implementation of the link. It specifies if it has to call exiftool.

#ftpMinPeriod: Define a minimum time limit to upload generated picture. (in seconds)
(! Be carefull because, it can skip some pictures)
