Content Manager Service (CMS)

HTTP Input

See config file to configure the http listener.

Test Url

http://localhost:port/command?key1=value|key2=value

We assume that separator is the character |

Commands

update

example: http://localhost/update?CURRENTARTIST=Shakira|MYTXT=you%20are%20listening%20to%20@@@CURRENTARTIST@@@


            
            update  sets variables of CMS which 
            are integrated in slides.

            This command allows recursion in the arguments. It means that you can define a 
            variable with oder variables by calling it in the format: @@@VARNAME@@@ (see 
            MYVAR above)

        

            broadcast

        

            example: http://localhost/broadcast?BSLIDE=webcam

        

            broadcast is used with the parameter BSLIDE

        

        

            updateandbroadcast

        
example: http://localhost/updateandbroadcast?BSLIDE=webcam|WEBCAMPIC=C:\webcam.jpg
http://localhost/updateandbroadcast?BSLIDE=nowplaying|CURRENTARTIST=Shakira

        

            updateandbroadcast sets variables and broadcast the 
            slide BSLIDE. 

        

            loadslidecart

        
example: http://localhost/loadslidecart?SLIDE1=onair|SLIDE2=clock|SLIDE3=ad1

        

            This command sets the list of slides which are automatically and periodicaly 
            broadcasted. The broadcast interval is defined in the config file.
