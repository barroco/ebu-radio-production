Content Manager Service (CMS)

Slides Canvas

A slide is the source file of a final picture which is send to receivers.
It gathers several components:

  • the text associated to the image (for receivers which display only text)

  • the link (url)

  • the canvas of the image

All these informations are stored in an xml file. The name of the file is the name of the slide and must be saved in the folder:

 %DataFolder%\img\slides\

DataFolder is defined in config file.

Structures

<?xml version="1.0" encoding="UTF-8"?>
<sequence><slide link="">your tags</slide></sequence>

The parameter link is the link of the slide which is send across RadioVIS. If it is empty, the slide is sent with the link specified in config file as radiovislink.

Tags

Rectangle

<background>

Possible parameters: top, left, right, bottom, width, height, color, opacity.

position and size: in pixels
colors : fill color of the rectangle
opacity: 0 to 100

 

Image

<img>

Possible parameters: top, left, right, bottom, width, height, src, opacity.

position and size: in pixels
src : Can be a filename or an url.  It may contain variable "@@@ONAIR_PIC@@@"
opacity: 0 to 100

 

Random Imagendom Image

<imgrdm>

Possible parameters: top, left, right, bottom, width, height, folder, opacity.

position and size: in pixels
folder: Path to the folder which contains the pictures to randomly select.
opacity: 0 to 100

 

Text

<text>
<text content="@@@STATIONNAME@@@" top="0" right="32" font-color="#EEEEEE" font-size="17" font-style="bold" opacity="70"/>

Possible parameters: top, left, right, bottom, width, height, opacity, font-color, font-size, font-style, font-family, text-align, content.

position and size: in pixels
opacity: 0 to 100
font-color: text color
font-size: in pixels
font-style: underline, bold, italic separated by semicolon.
text-align: left, right, center
content: Text (it may contain variables) ie: "ON AIR: @@@ONAIR_SONG@@@"
