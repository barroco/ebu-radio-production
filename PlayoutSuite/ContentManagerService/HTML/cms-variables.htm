Content Manager Service (CMS)

Variables

In your slides canvas you can use variables by calling them like this: @@@VARNAME@@@.

Default values are declared in the variables file.
Variables can be used in

  1. Text tag (content field)

  2. Image source (src field)

 

To update a variable, please check the input update command
